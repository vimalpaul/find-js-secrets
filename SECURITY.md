# Security policy

If you discover a security vulnerability in this project or a sensitive data exposure caused by using this tool, please **do not disclose it publicly**. Instead:

1. Create a private issue or contact the repository owner: https://github.com/Lamiv1311
2. Provide enough detail to reproduce or triage the issue.
3. We will respond within a reasonable timeframe and coordinate disclosure.
